Panduan Pengguna

Ikuti langkah-langkah berikut untuk menggunakan sistem monitoring tanah berbasis IoT:

  1. Login: Masuk ke sistem menggunakan akun yang telah terdaftar.
  2. Dashboard: Lihat ringkasan suhu dan kelembaban terkini.
  3. Realtime: Pantau data sensor secara langsung.
  4. Riwayat: Lihat histori data suhu dan kelembaban.
  5. Grafik: Analisis tren data dalam bentuk grafik.
  6. Export: Unduh data riwayat dalam bentuk Excel atau PDF.
  7. Logout: Keluar dari sistem saat selesai.

Pastikan perangkat IoT tetap terhubung dengan internet untuk memastikan data dapat dipantau secara real-time.
